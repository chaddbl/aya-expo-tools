O que é esta página

Monitora a saúde do media server (o computador que roda o conteúdo da expo).

  • GPU Temperatura — se passar de 75°C aparece alerta amarelo, acima de 84°C é crítico
  • CPU / RAM — uso do processador e memória. Acima de 90% é alerta
  • Timelapse — fotos das câmeras a cada minuto. Use o controle deslizante para navegar no tempo
  • Alertas — qualquer anomalia detectada durante o dia
Use o seletor de data para ver dias anteriores. Os gráficos mostram o histórico de 1 amostra por minuto. O cursor deslizante do timelapse sincroniza com os gráficos — ao mover o timelapse, os gráficos acompanham o horário.
