Como configurar dispositivos
  1. Adicionar — clique em "+ Adicionar" na categoria desejada
  2. Configurar IP — preencha o IP do dispositivo na rede da sala (ex: 192.168.0.101)
  3. Testar — use "Testar agora" para verificar se o dispositivo responde
  4. Posicionar — carregue a planta na aba "Espaço" e arraste o dispositivo para a posição correta, ou clique em "Marcar na planta"
  5. Salvar — clique no botão "Salvar" no topo direito para gravar todas as alterações
Projetores: precisam estar na mesma rede e com PJLink habilitado (porta 4352).
Câmeras: precisam de usuário e senha RTSP para acesso ao vídeo.
TVs: precisam de MAC para Wake-on-LAN. Use "Descobrir" para encontrar automaticamente.
Tomadas: controlam energia de projetores e áudio remotamente.

Como funciona o agendamento

O agendamento liga e desliga projetores e TVs automaticamente nos horários configurados, todos os dias.

  1. Ligar às — horário em que os projetores ligam e os vídeos das TVs começam a rodar
  2. Desligar às — horário em que tudo desliga no final do expediente
⚠️ Se o Chataigne estiver controlando os projetores, deixe o agendamento desativado para evitar conflito (os dois tentariam ligar/desligar ao mesmo tempo).

O agendamento usa o fuso horário configurado. Confira se está correto para a cidade da expo.

Rede e acesso remoto
  • RustDesk — ID para acesso remoto à tela do media server (suporte sem ir ao local)
  • IP do servidor — IP fixo deste computador na rede da sala (ex: 192.168.0.10)
  • Gateway — IP do roteador da sala (geralmente 192.168.0.1)
  • Escanear rede — descobre todos os dispositivos conectados com IP e MAC. Útil para encontrar câmeras e TVs novas
  • Contatos — telefones da equipe técnica para emergências
Se não souber o IP do servidor, abra um terminal e digite ipconfig. O IP aparece em "Adaptador Ethernet" → "Endereço IPv4".

Para que serve o log

O log é compartilhado com a equipe remota pelo Portal AYA. Quanto mais detalhado, mais fácil diagnosticar problemas à distância.
